第一章:抢劫

回到顶部
正值夏季，天气十分炎热，炽热的太阳高挂在空中，毫不留情的照耀着大地。在繁华的星海省中，都在下着大雪，而作为星海省中最繁华的城市的星源市也不例外。虽然这里正下着鹅毛大雪，但是在从林峰山通往星源市的公路上，来来回回的车子依然不绝如缕。天空中的大雪还在不停的四处乱飞，好像要把全世界都掩盖住一样。 　　可是，在车水马龙的公路上，还站着一个少年，他个头不高，大概有一米七，长的偏瘦，感觉像是几天没吃过饭一样，他上身着一件旧T恤，下身穿一件黄的不能再黄的牛仔裤，脚上穿着一双旧拖鞋。他在不停的张望着，好似在寻找什么东西似的。在这寒冷的公路上，有不少穿着厚厚衣服的路人都用奇怪的眼神看着他，可是，他却浑然不自知自己已成为路人指点的对象，依然在那里张望着。 令人奇怪的是，大雪仿佛没有看到少年一般，少年的身上，脚边一丝雪花都看不到，哪怕雪花飞到他的身边，也好像受到了阻力一般散开。 　　一位中年妇女带着她的女儿正要从是少年身边走过，突然迎面飞来一辆摩托，摩托上的是一位带着黑帽子的强壮男人，他经过妇女旁边时顺手就拐走了妇女的皮包，然后嗡的一声，摩托飞驰而去。 　　中年妇女先是呆了一呆，然后才反应过来，接着开始大叫：“抢劫了啊，救命啊。”中年妇女用求助的眼神向四周的人们求助，可是众人却仿佛好像害怕受到连累一样，闪到了一边。 　　中年妇女开始失望了，然后拿起电话准备打110，可是正当她拿起电话时，只觉眼前忽然一闪，一个人影一闪而过，向着摩托消失的方向跑去。虽然人影比摩托慢了一步，但是人影的速度却比摩托还快，一眨眼间，众人还没有反应过来时，人影便回来了，并且手上还拿着一个皮包。 　　众人看清楚了，都惊讶不已，因为这个人影不是别人，他就是这个少年，少年拿着皮包递给了中年妇女：“阿姨，看看这是不是你的皮包。” 　　中年妇女接过皮包，仔细的检查了一番后，两眼含泪的表达了感激：“小伙子，谢谢你，非常谢谢你，你真是个好人。” 　　少年只是呵呵一笑：“阿姨言重了，区区小事，何足挂齿。阿姨快回家吧，免得再被抢了。” 　　中年妇女这才反应过来，又向少年鞠了一躬：“是是是，我们马上回家。”说完牵着小女孩就走了。 　　不远处的一辆白色宝马里，一个年迈中旬的中年男人感叹了一句：“不愧是林先生请来的保镖啊，如此年轻，实力便如此深不可测，速度也非常的惊人，而且心底还很善良，此人以后必成大器。难怪林先生常说人不可貌相啊。” 本来中年男人对少年感到一轻视，认为少年也算不上什么厉害的人物，但经过少年这一手，中年男人顿时心服口服。 　　见时机差不多了，中年男人下了车，向少年走过来。 “李先生您好，林先生由于比较忙，就让我来接您。先自我介绍一下，我叫王钟，是林家的管家。很高兴认识你，先请上车，我带您去林家。” 　　少年惊讶的看了一眼王钟，随后上了车，王钟开始发动车子，车子急驶而去。 　　在宝马身后，一辆破损的摩托车停在那里，摩托车的主人正是刚才那个带黑帽子的强壮男人，他的身上满是雪，而他的双眼却迸射出浓浓的寒意，正盯着宝马，好似在说前面的宝马死定了。 　　他拿出电话拨了个号码，嘟的一声响了，那边接起了电话，男人的表情立刻变得谄媚：“乔少，我们的计划被一个臭小子打乱了，您看，我们是不是派人给他松松筋骨？” 　　那边陷入了沉静。 过了几秒，男人见没有回应，以为对方没有听清楚，正要重述，一个阴沉的声音赫然响起：“你做主就行，别把人打死了，我倒要看看是何方神圣敢破坏我们的计划？至于那个女人，给我继续盯着，不要让她跑了。” 　　“是是是。我马上去办。” 　　接着男人又拨了个电话，喊来了人，似乎准备大干一场。而少年浑然不知自己陷入了一场巨大的阴谋当中。 　　车子正在行驶中，少年四处看看，又伸出手四处摸摸，感叹了一句：“原来这就是宝马啊，也没什么好了不起的，死老头还跟我说坐一下这车可以让我舒服一阵，也没觉得让我多舒服啊。” 　　王钟听到少年的感叹呵呵一笑：“李先生见笑了，宝马也只是一种交通工具而已，其实没您想的那么好。”少年也一笑，又坐在那里闭目沉思了，王钟见状也不说话，继续开着他的车。 　　天空还在下着雪。突然，王钟停了下来，原来是堵车了。前前后后十余辆车堵在一起，王钟见状，有些无奈：“李先生，抱歉了，前面堵车，要等一会儿呢。” 　　少年笑着回应：“钟叔莫对我客气，我叫李强，您叫我小强便可，既然堵车了，那就等会儿吧，我也不……” 　　少年也就是李强正要说不着急，却忽然双眼一眯，眼里迸射出寒芒，因为刚才被他修理的那个男人不仅找了十来个人，而且还抓来了那位中年妇女和她身边的小女孩，十个人拿着铁棍，从前方几百米出向他们这里走来，许多路过的司机看见这些人都心惊胆战，生生让开了路，怕惹上了麻烦，造成堵路的原因竟然是他们十几俩豪车堵在了路边，路过的司机也是敢怒不敢言。 他们一个个凶神恶煞的样子，好像要吃人似的，走到了宝马前面，围住了宝马一动不动。李强正欲下车，王钟却拦住了李强，见李强有些疑惑的看着他，才笑着解释：“这点小事怎能让李先生解决，交给我吧。” 　　“好吧！那钟叔小心一点。” 　　王钟下了车，径直向那十个人走去。被李强修理过的强壮男人见出来的不是那个少年,而是一个中年男人，眉头皱了皱：“奉劝阁下不要多管闲事，否则我不介意给阁下放放血。我们要找那个臭小子，阁下最好还是把他交出来，否则别怪我们不客气。” 　　王钟淡淡一笑：“阁下好大的口气，在这星源市还没有我王钟解决不了的事，我倒要看看，你会怎么给我放血。” 　　强壮男人闻言脸色剧变，因为他听说过王钟的大名，也许他不怕王钟，但却怕王钟身后的林家，那可是连他家少爷都忌惮的林家啊，要不然他家少爷也不会为了得到林家而追林家的千金林莉莉呢。 　　男人沉默了好一会儿,才开口：“阁下一定要管此事？” 　　王钟听后却像看白痴似的看着他：“我若不管，我还下车干嘛？你是白痴吗？” 　　男人听后一阵脸色一会儿红，一会儿白，见王钟没有一丝退让的动静，只好挥了挥手：“好，放开她们，我们走。” 在离开之前，强壮男人恶狠狠的看了宝马一眼：“别以为有林家罩着你，我就拿你没办法，咱们走着瞧！” 手下放开了中年妇女和小女孩，然后他们就走了。 　　得救的中年妇女忙跪在地上磕头道谢，王钟扶起中年妇女：“到我车上来吧，我送你们回家。”中年妇女和小女孩上了车。 　　李强惊讶王钟这么快就解决了，顿时对林家产生了好奇。

第二章.初到林家

那些人离开之后，车子终于不堵了，王钟发动了车子，继续向前行驶。他转头问中年妇女：“你的家在哪儿，我先送你们回去。”中年妇女说了一个地名，王钟忙开着车朝那个方向疾驶而去。 　　时间很快就过去了，中年妇女也回到了家，一下车就忙着道谢，王钟却只是淡淡一笑：“以后出门小心点。”就继续开车走了。 　　一个小时后，车子停在了一栋别墅门前，王钟转头一笑：“李先生到了，这就是林家了，请下车。” 　　李强点了点头，下了车，却忽然转身：“钟叔不必如此客气，叫我小强就行呢。” 王钟听后呵呵一笑：“好吧，那我就托大叫你小强了，请先等我把车停在车库里，然后带你进去。” 　　几分钟后，王钟带着李强进了别墅里，却看到一个靓丽的美少女，穿着睡衣，坐在沙发上看《喜羊羊与灰太狼》。电视里正放着灰太狼的经典话语：“我一定会回来的”。听到开门声，少女蓦然回头，却看到钟叔带着一个邋里邋遢的少年进屋，突然大喝一声：“站住，你是哪家的乞丐，敢进我家，给我滚出去。” 　　钟叔听后向美少女解释：“小姐，他不是乞丐，他是林先生请来保护你的，林先生有事，就没有带他来。” 　　却不想美少女听后有些生气：“就他这样的乞丐，长的又不高大又不威武，能保护的了我吗？我要告诉爹地，把他辞了。” 　　少年听后，看了一下自己全身，却没有觉得有什么不妥，只好呵呵一笑：“这位美女，首先申明一下，我不是乞丐，再说，你怎么知道我就不能保护你呢。” 　　少女一听，顿时就生气呢，说：“我不要你保护，我一定会告诉爹地把你辞了的。”说着就关了电视，穿上鞋，蹦蹦跳跳的到她的房间里去呢。 　　钟叔笑了笑对少年说道：“小强不要在意，小姐他就这样，从小被宠坏了，有点任性，希望你多担待点。” 　　少年听了后，忙笑道：“钟叔多虑了，我不会跟小女孩一般见识的。” 　　钟叔带着少年进了一个房间，对少年说道：“小强，这就是你的房间了，我先出去了，你先收拾一下吧。” 　　少年忙回道：“谢谢钟叔了."钟叔回道：“不用客气了。”说完就出去了。 　　少年看了看，就跳上了床，在那里静坐着练功，他修炼的是一种心法，叫做《轩辕夺命法》。 　　当他闭上眼，他想起了老头对他说的话：“这个心法是上古遗迹留下来的，总共有10卷，每卷代表每一个层次，你如果把10层都修炼完了，那么你就是天下无敌呢。好好修炼吧，你现在最多只能算个打架高手，因为你才学完第一层，也就是一点武功而已。学到第5层，你就踏入了修仙境界，修仙有4个境界，分别是仙道，神道，人道，修炼者。而每个境界分为初期，初期巅峰，中期，中期巅峰，高期，高期巅峰，高期巅峰大圆满，每个境界实力都不一样，至于前面5层都是为修仙打基础的。学完5层，你的武功就达到了极限。也就是武道。现在世俗界大多都是学武道的，武道分为6个层次，分别是武道神期，武道仙期，武道魔期，武道妖期，武道鬼期，武道人期。而每一层有7个境界，分别为初级，初级巅峰，中级，中级巅峰，高级，高级巅峰，高级大圆满。你现在只是鬼期初级，要好好修炼，不然，你会受苦的。” 　　少年当时疑惑不已，问了老头一句：”那老头你现在是什么级别了？”老头却神秘一笑：“你以后就会知道呢。” 　　于是，少年开始了努力修炼。 　　美少女上了楼，进了她的闺房后，忙拿起她的小巧精致的手机给她爹地打电话过去，只听嘟的一声，电话响了，那边接起了电话。美少女佯装生气道：“爹地，你都给我找了个什么保镖啊，我觉得他不好，你把他辞了吧。” 　　那边听了后，沉默了一会儿，然后说道：“莉莉，你以后就会明白的。” 　　美少女听后撒娇道：“不嘛，爹地，我听不明白你说的话，你就把他辞了好不好啊。” 　　那边停留了一会儿，说道：“我现在忙着开会，就不说了。等晚上我下班回来再说，好吗？我先挂了，就这样，拜拜。”说完就嘟的一声挂断了电话。 　　美少女气的大叫：“爹地，你怎么这样啊？”可惜那边久久没有回应。 　　美少女气得把手机扔在床上，然后下了床，自演自语道：“既然你不走，那我就整死你。”她拿着2瓶二锅头下了楼进了少年的房间。 　　少年听的脚步声，急忙收敛了气息，坐在了床上。 　　少女一进来就对少年说道：“喂，既然你不走，那我们打个赌，只要你把2瓶二锅头喝完且不醉的话，那我就允许你留下来，否则，你就给我卷起行李走人，不对，你没拿行李来，应该是直接滚人。你不答应也给我走人，否则，否则。。。。。” 　　少年却戏谑一下她：“否则你会怎样。” 　　少女却恐吓一笑：“否则姑奶奶就切了你。”说完还得意一笑。 　　少年却立即答道：“那要是我赢了，你能给我什么？” 　　少女却笑道：“你赢了姑奶奶就同意你留下来我不说了吗？” 　　少年摇了摇头，戏谑一笑，说道：“不行，你这横竖都是我吃亏，我赢了，你要给我一样东西。” 　　少女霸道的说道：“不行，在这里是我说了算，你不能提条件。” 　　少年立即答道：“你不答应，我也不答应你。” 　　少女说道：“不行。反正就是不行。” 　　少年也说道：“那我也不干。有种你就来切了我啊。” 　　少女却只能说：“你。。。。。”停顿了一会儿又说道：“那好吧，你先说你想要什么东西。只要不是太贵的，我都可以给你。” 　　少年神秘一笑，道：“放心，这东西不要钱，只要你同意就行。” 　　少女却恍然不知自己掉入了大灰狼的陷阱中，忙答道：“好吧。本姑奶奶就答应你，你先把二锅头喝完再说。” 　　少年说道：“那就这么说定了。”说完就拿起一瓶二锅头，咕咚一声喝了下去。 　　少女惊讶无比的看着他，说道：“你没醉？” 　　少年却神秘的说道：“你说呢。记得我的东西哦。”说完又拿起一瓶二锅头，只听咕咚一声，少年又喝完一瓶，而且还神清气爽的站了起来，对少女说道：“好呢，现在该兑现你的诺言了吧。” 　　少女垂头丧气的说道：“本姑奶奶同意你留下来呢，说吧，你要什么东西。” 　　少年说道：“我暂时还想不起来，等以后想起来再向你要。” 　　少女却得意一笑：“想不起来，那就没有了，以后也不会有。”说完便跑出了房间。 　　少年无奈一笑，又继续坐在床上练功。
